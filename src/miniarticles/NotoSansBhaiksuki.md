
# Noto Sans Bhaiksuki

Noto Sans Bhaiksuki is a sanserif design intended for general use for the Bhaiksuki script.

It contains 863 glyphs and 9 OpenType features, and supports 103 characters from the Unicode blocks: Bhaiksuki.
